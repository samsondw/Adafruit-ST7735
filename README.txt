This is an mbed library for several Adafruit displays based on the ST7735 driver.

These displays use SPI to communicate, 4 or 5 pins are required to interface (RST is optional).

Adafruit invests time and resources providing open source code, please support Adafruit and open-source hardware by purchasing products from Adafruit!

Also requires the Adafruit_GFX library.
