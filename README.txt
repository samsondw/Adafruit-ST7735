This is an mbed library for several Adafruit displays based on the ST7735 driver.

  This is a library for the Adafruit 1.8" SPI display.

  The 1.8" TFT shield ----> https://www.adafruit.com/product/802
  The 1.44" TFT breakout ----> https://www.adafruit.com/product/2088
  as well as Adafruit raw 1.8" TFT display ----> http://www.adafruit.com/products/618
